## Navigation + View
- Tab | Tab | Move between fields, options, and buttons.
- Esc | Escape | Deactivate the selected item or get out of a mode.
- Ctrl + Shift + H | Help Menu | Open How to Use This App (expectations + FAQ).
- Ctrl + Shift + → | Next Card | Move selection to the next card (wraps).
- Ctrl + Shift + ← | Prev Card | Move selection to the previous card (wraps).
- Ctrl + Shift + 1-0 | Target Card | Select card 1, 2, 3, 4...10.
- Ctrl + Shift + V | Toggle View | Cycle Notes -> Split -> Map.
- Ctrl + Shift + G | Go to Location | Focus map on selected note/card location.
- Ctrl + Cmd + Shift + V | Toggle Tab | Cycle Dispatch -> Alert -> Summary/Export -> Settings -> About / Legal.

## Create New
- Ctrl + Shift + N | New Note | Create a new note in the selected card and focus it.
- Ctrl + Shift + C / Ctrl + Cmd + Shift + N | New Card | Create a new card.

## Set Location
- Ctrl + Shift + L | Set Location | Tag the selected card or note with a location.
- Ctrl + Cmd + Shift + L | Drop Pin Location | Start drag-pin location tagging for the selected card or note.

## Search
- Ctrl + Shift + S | Search | Open search in Collect mode.

## Card Actions
- Ctrl + Shift + R | Report | Toggle Report on the selected note.
- Ctrl + Shift + M | Min Card | Minimize the selected card.
- Ctrl + Shift + P | Card Color | Toggle through card color options for the selected card.
